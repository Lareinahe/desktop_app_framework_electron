## 简介

介绍跨平台桌面应用框架Electron的基本概念、基本原理、核心优势以及额外特性；介绍了有哪些用Electron开发出的常见应用；解释了如何安装Electron；以及如何上手使用；以及如何打包和部署；总结了相关的心得，比如asar、对Python的支持、如何继承js的框架，最后附上相关文档和教程等资料。
